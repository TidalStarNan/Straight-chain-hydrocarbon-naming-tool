# 非环烃命名工具

一个用于生成非环烃（包括直链和支链烷烃、烯烃、炔烃）系统命名的小工具。

## 项目简介

这是一个实用的化学工具程序，由编程初学者逐步完善而成。它实现了有机化学中非环烃类化合物的基本命名规则，支持带支链结构的烃类命名。

## 主要作用

该程序的核心功能是：**根据用户输入的碳骨架结构（包括支链信息），自动输出对应的非环烃系统名称**。

具体来说，它可以：

1.  **识别烷烃**：输入碳数和支链信息，输出如"甲烷"、"2-甲基丁烷"、"3-乙基-2,4-二甲基己烷"等名称。
2.  **识别烯烃**：输入碳数、双键位置和支链信息，输出如"乙烯"、"1-丁烯"、"3-甲基-2-戊烯"等名称。
3.  **识别炔烃**：输入碳数、三键位置和支链信息，输出如"乙炔"、"1-丁炔"、"4,4-二甲基-2-己炔"等名称。

## 程序特点

- **规则化**：程序内部嵌入了国际纯粹与应用化学联合会（IUPAC）的非环烃命名规则。
- **简单直接**：用户交互清晰，输入必要信息即可获得标准化的命名结果。
- **学习工具**：对于化学学习者，它可以作为验证命名规则理解和快速查阅的辅助工具。

## 关于"新手程序"的说明

请理解，这是一个在**持续学习和改进**中的程序。

- **当前能力**：已能处理**带支链的非环烃**命名，包括选择主链、编号优先级判断和多重取代基标注。
- **核心目的**：本项目在实践中学习并实现了以下编程概念：
  - 条件判断与优先级决策逻辑
  - 字符串处理和格式化输出
  - 函数封装与模块化设计

## 未来可能的改进

随着开发者技能的提升，这个程序未来可以扩展：
1.  处理多个官能团（如同时含羟基、羰基的烃类衍生物）。
2.  支持立体化学标记（如Z/E、R/S构型）。
3.  提供图形化界面，支持可视化结构绘制。
