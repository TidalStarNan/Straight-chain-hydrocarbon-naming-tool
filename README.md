# 直链烃命名器

一个用于根据碳原子数生成直链烃（烷烃、烯烃、炔烃）系统命名的小工具。

## 项目简介

这是一个简单的**新手程序**，由编程初学者为了练习基础编程概念而创建。它实现了有机化学中直链烃的基本命名规则。

## 主要作用

该程序的核心功能是：**根据用户输入的碳原子数量和官能团类型，自动输出对应的直链烃系统名称**。

具体来说，它可以：

1.  **识别烷烃**：输入碳数，输出如"甲烷"、"乙烷"、"癸烷"等名称。
2.  **识别烯烃**：输入碳数，输出如"乙烯"、"丙烯"、"辛烯"等名称，并能处理需要标明双键位置的情况（例如"1-丁烯"）。
3.  **识别炔烃**：输入碳数，输出如"乙炔"、"丙炔"、"戊炔"等名称，并能处理需要标明三键位置的情况（例如"1-丁炔"）。

## 程序特点

- **规则化**：程序内部嵌入了国际纯粹与应用化学联合会（IUPAC）的直链烃命名规则。
- **简单直接**：用户交互简单，只需提供最少的信息即可获得结果。
- **学习工具**：对于化学学习者，它可以作为一个快速查阅直链烃名称的辅助工具。

## 关于"新手程序"的说明

请理解，这是一个为**学习和练习**而创建的程序。

- **局限性**：它目前**只能处理无支链的直链烃**。对于含有复杂支链、多个官能团或环状结构的有机物，它无法正确命名。
- **核心目的**：本项目的首要目的是实践以下编程基础知识，而非构建一个功能完备的化学软件：
  - 变量与数据类型
  - 条件判断（if/else）
  - 循环
  - 数组/列表的使用
  - 基本的输入/输出操作
  - 函数封装

## 未来可能的改进

随着开发者技能的提升，这个程序未来可以扩展：
1.  处理带支链的烃类。
2.  处理多个官能团。
